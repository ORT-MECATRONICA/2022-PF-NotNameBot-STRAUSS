# NotNameBot
ROS based nameless robot.

## ROS important commands (falta)
* roscore
* rosrun
* rviz
* rosnode / rostopic
* rosrun rosserial_arduino serial_node.py /dev/ (Your serial port)
* rostopic echo / (Your topic)
